# 󰣇 My Arch Linux Dotfiles

Добро пожаловать в мой конфиг! Здесь я храню настройки своей системы, чтобы ничего не потерялось.

## 🖥️ System Info
* **OS**: Arch Linux
* **WM**: Sway (Wayland)
* **Shell**: zsh
* **Browser**: Floorp
* **Terminal**:Foot

## 🛠️ Особенности
* **Wallpaper**: Скрипт `random_wall.sh` для живых/рандомных обоев.
* **Status Bar**: Настроенный Waybar.
* **Backup**: Управление через GNU Stow.

## 🚀 Как установить
1. Склонировать: `git clone https://github.com/Atom111minecraft/dotfiles.git`
2. Зайти: `cd dotfiles`
3. Применить: `stow sway waybar`
